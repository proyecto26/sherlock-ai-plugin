# with-formulas

Includes formula images with detailed explanations

## Usage

- Insert formula images extracted by MinerU
- Explain each key formula in detail
- Clarify symbol meanings and computation steps
- Describe the intuitive meaning behind each formula

## Example

```markdown
![](images/formula_1.jpg)

This formula defines how the memory vector is constructed:
- $e_t$ is the final memory vector
- Results from different n-gram lengths and hash heads are concatenated
- This design allows table capacity to scale independently
```
