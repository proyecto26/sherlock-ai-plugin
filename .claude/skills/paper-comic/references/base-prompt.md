# Basic Guidelines for Academic Paper Comic Generation

## Image Specifications

- **Type**: Comic page, multi-panel
- **Orientation**: Portrait
- **Aspect ratio**: 2:3
- **Resolution**: High definition, with clearly readable text

## Panel Structure

### Panel Borders
- Black border lines (1–2px)
- White spacing between panels (8–12px)
- Clear reading order (left to right, top to bottom)
- Varying panel sizes to create visual rhythm

### Panel Composition
- Each panel has a clear focal point
- Proper use of foreground, midground, and background
- Diverse camera angles: eye-level, top-down, low-angle, close-up, wide shot
- Actions and gaze guide the reading flow

## Text Elements

### Speech Bubbles
- Oval-shaped with a pointer toward the speaker
- White fill with black outline
- Hand-drawn style font

### Narration Boxes
- Rectangular boxes placed along panel edges
- Light yellow or beige background
- Used for explanations, time jumps, and background context

### Thought Bubbles
- Cloud-shaped with small bubbles pointing to the thinker
- Used for inner monologue

### Text Guidelines
- **All text must be in English**
- Technical term format: English (Original term)
- Clear fonts with appropriate sizing
- Key terms may be bolded

## Scientific Concept Visualization

Transform abstract concepts into concrete visuals:

| Concept Type | Visual Representation |
|-------------|----------------------|
| Neural networks | Glowing nodes and connecting lines |
| Data flow | Light particles moving along paths |
| Algorithms | Geometric shapes, building blocks |
| Memory / storage | Bookshelves, filing cabinets, libraries |
| Search / retrieval | Magnifying glasses, spotlights |
| Learning / training | Climbing, growing plants |
| Innovation / breakthrough | Bursts of light, breaking out of shells |

**Principles**:
- Use metaphors instead of formulas
- Use diagrams instead of text-heavy explanations
- Make abstract concepts “visible”

## Character Depiction

### Mentor Character
- Wise and approachable expressions
- Natural hand gestures when explaining
- Can point to diagrams or a blackboard

### Student Character
- Curious and focused expressions
- Eyes light up when asking questions
- Moments of sudden realization when understanding

### Interaction Scenes
- Natural, non-stiff dialogue
- Eye contact between characters
- Expressive body language

## Language Requirements

- All dialogue: English
- All narration: English
- Technical terms: English (Original term)
- Labels and annotations: English
- Standard English punctuation: "", . ! ?
