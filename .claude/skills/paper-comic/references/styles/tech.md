# Tech Style

A modern, digital aesthetic, suitable for AI, computer science, and engineering papers.

## Line Work
- Precise geometric feel (2px)
- Circuit-pattern accents
- Glowing edge effects

## Character Design
- Modern attire
- Focused expressions
- Optional wearable tech accessories

## Background
- Digital elements, screens, code
- Grid patterns
- Glowing interface elements

## Color Palette
- Primary colors: cyan #00D4FF, deep blue #1A365D
- Accents: neon green #00FF88, electric purple #805AD5
- Background: dark gray #1A202C

## Applicable Use Cases
AI, machine learning, computer science, engineering and technology
